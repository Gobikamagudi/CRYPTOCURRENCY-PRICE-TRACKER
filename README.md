\# Crypto Price Scraper using Selenium



This project scrapes real-time cryptocurrency prices from LiveCoinWatch using Selenium WebDriver.



\## Features

\- Scrapes top 10 cryptocurrencies

\- Captures website screenshot as proof

\- Saves data to CSV

\- Handles JavaScript-based websites



\## Tech Stack

\- Python

\- Selenium

\- Pandas

\- WebDriver Manager



\## How to Run

```bash

pip install selenium webdriver-manager pandas

python main.py
